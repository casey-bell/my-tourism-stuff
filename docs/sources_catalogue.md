Sources Catalogue
Record all datasets used in the project: ownership, access URLs, licences, refresh cadence, and retrieval method. Update this catalogue whenever a source changes or a new dataset is added.

How to use this catalogue
Add a row for each dataset you acquire into data/raw/ (or via DVC remote).
Confirm the licence and include any attribution requirements.
Record the update frequency (e.g. Monthly, Quarterly, Annual, Ad hoc).
Note the retrieval method (download, API, manual, DVC).
Update Last Verified after each successful pipeline run.
